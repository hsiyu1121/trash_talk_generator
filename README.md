# 幹話產生器

使用Node.js, Express, Express-handlebars 套件製作而成

## 功能清單
* 點選每個職業有相對應的幹話
* 點選"Trash Talk Generator"即可重新整理
* 一直點選"說幹話"即可不斷產生幹話

## 環境需求
* Node.js: v10.15.0
* express: v4.17.1
* express-handlebars: v5.1.0

## 啟動方式
* 將專案下載至本機內

  ``git clone https://github.com/hsiyu1121/trash_talk_generator.git``
* 切換至資料夾內

  ``cd trash_talk_generator``
* 安裝相關的套件

  ``npm install``
* 執行程式

  ``npm run start``
* 開啟瀏覽器輸入以下網址

  ``http://localhost:3000``
